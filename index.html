Power BI Análise de dados Dashboard interativo
Análise Climática — Nordeste do Brasil
Dashboard interativo criado para comparar indicadores climáticos entre cidades do Nordeste, com KPIs, ranking, mapa, linha temporal e insight analítico.
Desafio Consolidar informações climáticas de diferentes cidades em uma leitura comparativa e simples.
Solução Estruturação de um dashboard visual com métricas principais, comparação entre cidades e análise complementar.
Resultado Visualização rápida dos padrões climáticos e melhor compreensão das diferenças entre os cenários analisados.
